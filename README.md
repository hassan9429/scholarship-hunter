# 🎓 ScholarshipHunter Pro — Pakistani Students Edition

A professional scholarship discovery web app for Pakistani students seeking fully funded MS/MPhil/PhD abroad.

## 🚀 Deploy on Streamlit Cloud (Free — 5 minutes)

### Step 1: Upload to GitHub
1. Create a free account at [github.com](https://github.com)
2. Click **New Repository** → name it `scholarship-hunter`
3. Upload these files:
   - `app.py`
   - `requirements.txt`
   - `.streamlit/config.toml`
   - `README.md`

### Step 2: Deploy on Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with your GitHub account
3. Click **New app**
4. Select your `scholarship-hunter` repository
5. Main file: `app.py`
6. Click **Deploy** ✅

Your app will be live at:
`https://your-username-scholarship-hunter-app-xxxx.streamlit.app`

**Share this link with any student — no login, no install, works on mobile.**

## Features
- 35+ scholarships across Europe, Middle East, Australia, Asia, Canada, Pakistan
- Auto-projected future deadlines (never expired)
- Filters: Region, Field, Degree, Language, IELTS band, HEC route, IELTS waiver
- Hidden Gems spotlight (low-competition scholarships)
- Deadline Calendar with urgency color coding
- Export to CSV / Excel
- Dark mode professional UI
